CITAHub Docs Versions

CITA 有一个自己的发版节奏，具体信息可以查看我们的开发路线图（即将着陆）

  • 如果对 CITA 最新引入的改变感兴趣并且想提些建议和反馈，可以使用我们的最新源码编译。
  • 如果要部署到生产环境，我们强烈建议您使用我们的已发版的稳定版本。

请注意：工具链的版本和 CITA 的版本号是不一致的。但是请不要担心，因为在此文档中，当切换 CITA 版本时，我们都给出的是与其适配的 CITA 工具链版本，根据提示下载相应版本进行使用就可以了。
