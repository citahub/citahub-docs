CITA 社区

我们旨在打造连接用户，开发者，运营方的闭环社区。

如果你想寻求帮助

去哪里寻求帮助取决于你需要什么样的帮助。

如果你是个人开发者，想要寻求技术上的支持

CITAHub Talk 将是你最好的选择。先搜索论坛里是否已有相同的问题，不论你遇到的是代码层面的问题还是使用时候的困扰。

如果你是开发团队、企业组织，想要寻求行业合作或解决方案

请申请加入 我们的社区.

区块链是用来创造价值网络的工具，基于该理解，我们希望 CITAHub 能够联合各方资源，为开发价值网络提供场景、开发工具及最佳实践。我们可以提供以下帮助：

  • 为开发者团队提供基于 CITA 的底层链开发以及应用层开发咨询服务
  • 为企业用户提供业务层系统集成服务，将 CITA 与已有业务系统深度整合。针对不同行业由合作伙伴提供一揽子解决方案。将区块链不可替代的价值与现有业务逻辑深度整合，帮助资源优势方迅速构建平台壁垒

如果你想参与贡献

阅读 贡献指南 首先。

无论是代码还是文档，都非常欢迎大家参与贡献。

通常地, CITA 使用:

  • CIP（CITA改进提案）来管理 CITA的协议
  • Github Issue 和 Github Project 来创建、组织和跟踪 Issue，包括各种 Bug 和 功能改进
  • CITAHub Docs for documentation
  • Github Pull Request 来检视和合并代码或者文档的改进
